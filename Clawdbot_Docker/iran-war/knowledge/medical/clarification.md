# Clarification Rules

# 

# Purpose: Ask the minimum necessary follow-up questions when medical information is insufficient.

# 

# Ask only what changes triage

# 

# Priority questions:

# 

# duration: when it started, how long it has lasted

# 

# severity: mild / moderate / severe

# 

# associated symptoms: symptoms occurring together

# 

# medication history: current medications or recent changes

# 

# chronic diseases: underlying disease, pregnancy status

# 

# Symptom question template

# 

# When did it start?

# 

# Is it getting worse or staying the same?

# 

# Are there symptoms like fever, chest pain, shortness of breath, or fainting?

# 

# Are you taking any medications, or were any recently changed?

# 

# Do you have any existing medical conditions?

# 

# Medication question template

# 

# What is the exact name of the medication?

# 

# Is the unit written as mg or mcg?

# 

# Who prescribed it?

# 

# When did you start taking it and is there a scheduled follow-up test?

# 

# Are you taking any supplements or other medications with it?

# 

# Stop asking rule

# 

# If a red flag is already present, give emergency guidance before asking more questions.

# 

# If enough information is already provided, minimize additional questions.
