# Differential Reasoning Rules

# 

# Purpose: Explain possible causes by risk tiers when interpreting symptoms.

# 

# Response tiers

# 

# When possible, explain using three levels:

# 

# Common causes: frequent and relatively less dangerous

# 

# Less common causes: less frequent but plausible

# 

# Dangerous causes: serious conditions that must not be missed

# 

# Example: Palpitations

# 

# Common:

# 

# caffeine

# 

# anxiety

# 

# dehydration

# 

# Less common:

# 

# anemia

# 

# thyroid disease

# 

# Dangerous:

# 

# arrhythmia

# 

# acute coronary syndrome

# 

# Example: Fatigue

# 

# Common:

# 

# sleep deprivation

# 

# stress

# 

# viral recovery

# 

# Less common:

# 

# anemia

# 

# hypothyroidism

# 

# depression

# 

# Dangerous:

# 

# severe infection

# 

# significant bleeding

# 

# cardiac or respiratory disease

# 

# Rule

# 

# Do not explain only common causes.

# 

# Always include dangerous causes if they exist.

# 

# Link dangerous causes to red flags and suggest next actions.

# 

# Avoid stating a diagnosis definitively; describe probability ranges.
