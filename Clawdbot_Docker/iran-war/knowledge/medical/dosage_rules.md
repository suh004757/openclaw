# Dosage Safety Rules

# 

# Purpose: Prevent unsafe medication-dose advice.

# 

# Safety rules

# 

# The model must not:

# 

# prescribe medications

# 

# recommend dose changes or titration

# 

# suggest starting a prescription drug

# 

# recommend abruptly stopping a prescription drug

# 

# provide pediatric dosing

# 

# suggest off-label dosing as a decision

# 

# override a clinician’s prescription

# 

# Allowed actions

# 

# The model may:

# 

# identify the drug and ingredient

# 

# explain unit conversions (mg, mcg, μg, mL)

# 

# explain typical use and precautions

# 

# say a prescribed dose is commonly used if well-supported

# 

# advise confirming with the prescribing clinician

# 

# explain typical follow-up intervals as general information

# 

# Required wording for dose questions

# 

# “Arbitrary dose increase or reduction is not appropriate.”

# 

# “The prescribing physician’s judgment and follow-up results take priority.”

# 

# “The unit must be verified first.”

# 

# Special note for thyroid medication

# 

# Levothyroxine dosing errors often come from unit confusion.

# 

# 50 mcg = 0.05 mg.

# 

# Dose adjustment is typically done after follow-up lab tests.
