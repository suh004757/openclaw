# Medications Reference

# 

# Purpose: Provide a consistent structure for medication questions.

# 

# Standard response framework

# 

# Confirm drug name and ingredient

# 

# Confirm unit (mg, mcg, μg, mL)

# 

# Explain general use

# 

# Explain common usage principles

# 

# Warn against arbitrary dose changes

# 

# Mention need for follow-up or consultation

# 

# Unit notes

# 

# 1 mg = 1000 mcg

# 

# mcg and μg mean the same (microgram)

# 

# Correct units before answering dose questions.

# 

# Levothyroxine / Synthroid / Euthyrox

# 

# Ingredient: levothyroxine

# 

# Use: thyroid hormone replacement

# 

# Common strengths:

# 

# 25 mcg

# 

# 50 mcg

# 

# 75 mcg

# 

# 100 mcg

# 

# Key points:

# 

# 50 mcg = 0.05 mg

# 

# usually taken in the morning on an empty stomach

# 

# 30–60 minutes before food

# 

# separate from iron, calcium, magnesium, multivitamins, and coffee

# 

# Follow-up:

# 

# dose decisions usually made after TSH recheck in 4–8 weeks

# 

# Possible over-replacement symptoms:

# 

# palpitations

# 

# tremor

# 

# insomnia

# 

# anxiety

#
