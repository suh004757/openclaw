# Medical Red Flags

# 

# Purpose: Screen for emergency signals first.

# 

# Signals requiring immediate evaluation

# 

# severe chest pain

# 

# difficulty breathing or inability to speak due to breathlessness

# 

# loss of consciousness or new confusion

# 

# one-sided weakness or speech difficulty

# 

# severe allergic reaction (lip/tongue swelling, wheezing)

# 

# uncontrolled bleeding

# 

# severe dehydration with minimal urine

# 

# severe pain or bleeding during pregnancy

# 

# high fever with altered consciousness or neck stiffness

# 

# suicidal thoughts or self-harm risk

# 

# Same-day medical evaluation

# 

# fever lasting several days

# 

# worsening abdominal pain

# 

# repeated vomiting/diarrhea preventing hydration

# 

# severe dizziness, palpitations, exhaustion

# 

# sudden swelling or weight change

# 

# sudden severe headache

# 

# rash or swelling after medication

# 

# Situations suitable for outpatient follow-up

# 

# fatigue lasting weeks

# 

# mild weight change

# 

# cold intolerance

# 

# constipation

# 

# dry skin or swelling

# 

# abnormal lab values without acute symptoms

# 

# Response principle

# 

# Check red flags first.

# 

# If present, prioritize urgent guidance over explanation.

# 

# If absent, explain common causes and follow-up needs.

# 

# Avoid definitive diagnosis.
