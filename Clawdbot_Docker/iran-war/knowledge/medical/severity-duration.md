Severity and Duration Guide

Purpose: Quickly classify symptom seriousness.

Severity scale

Mild: daily activities possible but uncomfortable

Moderate: some impairment of daily function

Severe: daily activities difficult or emergency signs present

Duration guide

<24h: acute change, monitor worsening speed

1–7 days: possible infection, medication effect, temporary change

>2 weeks: outpatient follow-up becomes more important

Rule

Severe + red flag → emergency evaluation

Moderate + several days → same-day care or early outpatient visit

Mild + >2 weeks → outpatient follow-up recommended
